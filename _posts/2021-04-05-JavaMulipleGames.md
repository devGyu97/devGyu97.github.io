# Java Multiple Games



Java를 이용하여 구현한 세 가지의 GUI 게임이다.

게임으로는

* TicTacToe
* Omok
* 1 to 50

세 가지로 구성되어 있다.

해당 프로젝트는 3학년 시절 진행한 프로젝트로써 깃을 통한 배포 목적이 초기에 없었기에 가급적이면 첨부한 링크를 통해 이클립스에서 직접 Import 하여 보는 것을 추천한다.

간략한 알고리즘에 대한 설명과 개발 과정 중의 버전별 히스토리 역시 참고하길 바란다.
